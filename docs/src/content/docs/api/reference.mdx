## Authentication

<Tabs>
  <Tab label="TypeScript">
```typescript
import { configure } from "@superglue/client";

configure({
  baseUrl: "https://api.superglue.cloud/v1",
  apiKey: "your-api-key",
});
```
  </Tab>
  <Tab label="Python">
```python
from superglue_client import SuperglueClient

client = SuperglueClient(
    base_url="https://api.superglue.cloud/v1",
    token="your-api-key"
)
```
  </Tab>
</Tabs>
